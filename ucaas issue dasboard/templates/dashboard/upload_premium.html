CSV Requirements

  • Ensure your file contains a header row with meaningful column names.

  • Required Columns & Acceptable Names:

    • • Issue ID (ID, Number, No, Ticket Number)
    • • Component Name (Component, Section, Module)
    • • Issue Message (Error, Issue, Message, Bug Description)
    • • MCM Comments (Comments, Status, Fixed, Test, Remarks, Notes)
  • Optional columns: "Screenshot URL", "Capanicus Comment", "Planned Hours", "Utilized Hours", "Resource", "Priority"

  • Smart Mapping: System automatically detects columns like "ID", "Bug", "Section", etc.
